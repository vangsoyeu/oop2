Lớp File được kế thừa từ lớp Object lớp File được thực thi từ interface Serializable, Comparable Lớp File có mục đích là: tạo các file và thư mục, để tìm kiếm các file, xóa các file, … Lớp File gồm 4 phương thức khởi tạo: - File(File parent, String child): Tạo một phiên bản Tệp mới từ tên đường dẫn trừu tượng gốc và chuỗi tên đường dẫn con. - File(String pathname): Tạo một phiên bản Tệp mới bằng cách chuyển đổi chuỗi tên đường dẫn đã cho thành một tên đường dẫn trừu tượng. - File(String parent, String child): Tạo một phiên bản Tệp mới từ chuỗi tên đường dẫn gốc và chuỗi tên đường dẫn con. - File(URI uri): Tạo một phiên bản Tệp mới bằng cách chuyển đổi tệp đã cho: URI thành một tên đường dẫn trừu tượng. Phương thức thường dùng: - getName(): Trả về tên của tệp hoặc thư mục được biểu thị bằng tên đường dẫn trừu tượng này. + Tên phương thức: getName. + Tham số đầu vào: không có. + Giá trị trả về: là 1 chuỗi chứa tên của đối tượng File đã cho. - getPath(): trả về đường dẫn tuyệt đối của đối tượng File đã cho dưới dạng một chuỗi. + Tên phương thức: getPath. + Tham số đầu vào: không có. + Giá trị trả về: một chuỗi chứa đường dẫn tuyệt đối của đối tượng File đã cho.
